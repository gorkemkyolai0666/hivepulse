# CampLedger AI Sistemi

## Durum

CampLedger MVP'de AI özelliği bulunmamaktadır.

## Gelecek Planlar (Faz 3)

- Rezervasyon talebi otomatik yanıtlama
- Doluluk tahminleme modeli
- Sezonluk fiyat optimizasyonu önerileri
- Misafir iletişim otomasyonu

## Mimari Hazırlık

Backend modüler yapısı, gelecekte `ai` modülü eklenmesine izin verir. Mevcut veri modeli (rezervasyonlar, sezon fiyatları, doluluk) ML eğitimi için yeterli sinyal sağlar.
